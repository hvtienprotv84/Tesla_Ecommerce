# Firebase
- Tạo Database bằng Firebase (https://console.firebase.google.com/)
- `Lưu Ý: Cần tạo 2 mục này`
- 1. `Authentication`
- 2. `Firestore Database` 

```bash
const firebaseConfig = {
   apiKey: "",
   authDomain: "",
   projectId: "",
   storageBucket: "",
   messagingSenderId: "",
   appId: "",
   measurementId: ""
 };
```
